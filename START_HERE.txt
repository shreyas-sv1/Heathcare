🛠️ COMMON ISSUES & SOLUTIONS:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Python not found
   ✅ Install Python 3.7+ from python.org
   ✅ Add Python to PATH during installation

❌ MySQL connection failed
   ✅ Start MySQL server
   ✅ Check username/password in application.properties
   ✅ Ensure healthcare_db exists: SHOW DATABASES;

❌ Port 8080 already in use
   ✅ Change port in application.properties: server.port=8081
   ✅ Or stop the process using port 8080

❌ Build fails with Java errors
   ✅ Check Java version: java -version (must be 17+)
   ✅ Set JAVA_HOME environment variable
   ✅ Ensure all files were generated: python verify_build.py

❌ Frontend can't connect to backend
   ✅ Ensure backend is running: http://localhost:8080/api-docs
   ✅ Check browser console (F12) for errors
   ✅ Clear browser localStorage
   ✅ Verify CORS is enabled in SecurityConfig

❌ JWT token errors
   ✅ Login again to get fresh token
   ✅ Ensure jwt.secret is set in application.properties
   ✅ Check token format: Bearer <token> (with space)
